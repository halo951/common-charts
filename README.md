> 这个库适用于 vue2.x, vue3.x 的 echarts 图表组件过渡. 并且提供了一些常用图表参数模板.

-   support: vue >= 2.6 or vue2.7.\* or vue 3.x
-   chart template

    -   bar
    -   pie
    -   line
    -   ...
